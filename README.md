# glassBoard

An infinite^ canvas whiteboard with pins and images.
My front-end development task for CSI.

Deployed at: https://csi-task.vercel.app

## Features

- Infinite^ pannable canvas with grid overlay
- Pin notes with text
- Upload and place images
- Drag to reposition elements
- Delete pins and images

## Usage

Open index.html in a web browser or visit the deployed site.

### Controls

- Click and drag canvas to pan
- Pin button: Click to place a pin, click pin icon to toggle text box
- Image button: Upload and place images on canvas
- Grid button: Toggle grid visibility (hover or click to lock)
- Delete: Hover over elements to reveal delete button

### Pin Notes

- Click Pin button in toolbar
- Click canvas to place pin
- Click pin icon to open/close text box
- Drag pin icon to reposition
- Text box is resizable from bottom-right corner

### Images

- Click Upload image button
- Select image file
- Click canvas to place
- Drag image to reposition
- Hover to reveal delete button

## Requirements

Modern browser with ES6 support.
No build process or dependencies required.

## Deployment

Hosted on Vercel at https://csi-task.vercel.app
